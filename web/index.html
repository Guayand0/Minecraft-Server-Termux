Guia rapida y visual

Instala un servidor de Minecraft en Termux

Elige el tipo de servidor y la version, copia los comandos listos para pegar y sigue una guia pensada para que no tengas que adivinar nada. La version seleccionada se convierte automaticamente en la URL correcta para descargar el server.jar.

Selecciona tu servidor

Primero elige el servidor y luego la version. El comando de descarga se actualiza al instante.

Requisitos

Conexión a internet

+ 4GB de almacenamiento

+ 2GB de RAM

Descargar Termux desde f-droid, Google Play o Github

Información importante

El rendimiento de un servidor de Minecraft en un móvil es limitado. Este proyecto está diseñado para realizar pruebas o jugar con un número muy reducido de personas, de 1 a 3 jugadores, ya que el hardware de un dispositivo móvil no ofrece el mismo rendimiento que un PC o un servidor dedicado y puede tener problemas de lag.

En rendimiento, un servidor en Termux se compara aproximandamente con un servidor de Aternos, pero con diferentes limitaciones.

Instrucciones
Comando 1: instalar Debian en Termux Este comando prepara el entorno base donde luego vivira el servidor. 
curl -fsSL https://raw.githubusercontent.com/Guayand0/Minecraft-Server-Termux/main/1_install-Debian.sh | bash
Comando 2: descargar el servidor elegido La URL cambia segun el tipo de servidor y la version seleccionados arriba. 
curl -fsSL https://raw.githubusercontent.com/Guayand0/Minecraft-Server-Termux/main/2_install-MC.sh | bash -s "<URL_VERSION>"

Archivos

- Para crear un archivo usa touch antes del nombre del archivo. 

touch hola.txt

- Para modificar un archivo usa nano antes del nombre del archivo. 

nano server.properties

- Para guardar un archivo abierto usa Ctrl + O y luego Ctrl + X

- Para salir sin guardar un archivo abierto usa Ctrl + X y luego Y

- Para eliminar un archivo usa rm antes del nombre del archivo. 

rm hola.txt

- Para dar permisos de ejecución a un archivo usa chmod +x antes del nombre del archivo. 

chmod +x start.sh

Carpetas

- Para crear una carpeta usa mkdir antes del nombre de la carpeta. 

mkdir server_26.2

- Para entrar en una carpeta usa cd antes del nombre de la carpeta. 

cd server_26.2

- Para salir de una carpeta usa cd .. en la terminal.

- Para eliminar una carpeta y su contenido usa rm -r antes del nombre de la carpeta. 

rm -r server_26.2

- Para ver el contenido de una carpeta usa ls en la terminal.

Descargar

- Para descargar mods o plugins entra en la carpeta mods o plugins del servidor y usa curl -fsSL antes de la URL. Se descargará en la carpeta actual. 

curl -fsSL <URL_ARCHIVO>

Servidor de Minecraft

- Para iniciar el servidor usa ./start.sh dentro de la carpeta del servidor.

- Para parar el servidor usa stop en la terminal mientras el servidor este corriendo.

- Para reiniciar el servidor usa restart en la terminal mientras el servidor este corriendo.

- Para cambiar la cantidad de RAM usada por el servidor modifica el archivo start.sh y cambia el valor de RAM.
